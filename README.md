## Moment 是什么

Moment 是一款常驻 macOS 菜单栏的轻量级倒数日工具，来自国产独立开发团队 **Fireball 烧蛋工作室**（他们还做了 One Switch 那款工具）。

说白了，就是把你最在乎的那些日子，挂在你每天都要看的地方。抬头就能看到，不需要专门去打开什么 App。

它能做的事，其实比"倒数日"三个字要多一点：

- **倒数日**：婚礼、考试、旅行出发、还有多少天发工资……还没到的日子都能数
- **纪念日**：那些已经过去的日子，比如恋爱第几天、加入公司第几天，反过来从那天往后数
- **时间进度条**：今天已经过去了多少百分比？今年已经完成了多少进度？看着这个进度条，一种微妙的紧迫感会油然而生
- **年龄追踪**：加上家人朋友的生日，随时能看到他们几岁了，顺便提醒自己要常联系

每个事件都可以单独设置"常驻菜单栏"——也就是说，你最在乎的那个日子，可以直接摆在最显眼的位置，不用点开软件就看到。

## 跟手机倒数日 App 比有什么区别

有人可能会问：手机上不是有很多倒数日 App 吗，为什么要在 Mac 上再买一个？

区别就在于**你打开它的频率**。

手机上的 App，大多数人一周都不一定开一次，放在第三屏的文件夹里，靠的是通知推送。而 Mac 菜单栏是你每天工作时必看的地方，Moment 不需要你主动去找它，它就一直在那，日期信息自然而然就进了眼里。

被动提醒和主动查看，差距挺大的。
